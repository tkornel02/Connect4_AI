# Connect4_AI
 Implemented MiniMax algorithm in a console based Connect4.

## Launch 
```bash
git clone https://github.com/tkornel02/Connect4_AI.git
cd Connect4_AI
```
#### Compile 
```bash
javac -cp ./src .\src\Connect4Game.java -d ./out
```
#### Run

```bash
java -cp ./out Connect4Game
```
